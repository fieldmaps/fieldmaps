# Atlas Maps

[![atlas](/img/atlas.png)](https://atlas.fieldmaps.io/)

Currently, a map style based on global edge-matched subnational boundaries is being developed. A
preview of this style is available here: [atlas.fieldmaps.io](https://atlas.fieldmaps.io/).

Development priority is currently towards global map styles, however country-specific layers
remain available at URLs like the following: [atlas.fieldmaps.io/ssd](https://atlas.fieldmaps.io/ssd).
